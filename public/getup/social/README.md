# GETUP! Social Media Banners

This directory contains social media banners for GETUP!.

## Required Banners

- `banner-yt-2560x1440.png` - YouTube channel art
- `banner-tw-1500x500.png` - Twitter/X header
- `post-ig-1080x1350.png` - Instagram post

## Generation

See `/branding/getup/SOCIAL_BANNERS.md` for design specifications and generation instructions.

## Design Guidelines

- Background: `#0B0B0D` (near-black)
- Logo: GETUP! wordmark in white
- Accent: Red exclamation with subtle pulse effect
- Tagline: "No mercy. No snooze." bottom-right
- Typography: Bebas Neue or Anton (ultra-bold)
