# GETUP! App Icons

This directory contains all app icons for GETUP!.

## Required Icons

Place the following icons in this directory:

### iOS App Store
- `appstore-1024.png` (1024x1024)
- `ios-180.png` (180x180)
- `ios-167.png` (167x167)
- `ios-152.png` (152x152)
- `ios-120.png` (120x120)

### Android Play Store
- `android-512.png` (512x512)
- `android-192.png` (192x192)
- `android-144.png` (144x144)
- `android-96.png` (96x96)
- `android-72.png` (72x72)
- `android-48.png` (48x48)

### Favicons
- `favicon-16.png` (16x16)
- `favicon-32.png` (32x32)
- `favicon-96.png` (96x96)

### PWA Maskable
- `maskable-512.png` (512x512 with safe zone)

## Generation

See `/branding/getup/ICON_GENERATION.md` for instructions on generating these icons.

## Base Icon

Create a base 1024x1024 PNG with:
- Background: `#0B0B0D`
- Text: "GU!" in white (`#F5F7FA`)
- Exclamation: Red (`#FF2D2D`)
- Centered layout with safe margins

Then use the export script or manual generation to create all sizes.
